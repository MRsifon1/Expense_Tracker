# Expense Tracker

**Автор:** Иван Пожидаев
**Язык:** Python 3.x

## Описание программы

Expense Tracker — это простое кроссплатформенное приложение с графическим интерфейсом (GUI) для учёта личных финансов. Оно позволяет фиксировать ежедневные расходы по категориям и датам.

**Основные возможности:**
*   **Добавление расходов:** ввод суммы (с проверкой на положительное число), выбор категории и даты.
*   **Визуализация:** все расходы отображаются в удобной таблице.
*   **Фильтрация:** возможность отобразить расходы только по одной выбранной категории.
*   **Отчёты:** подсчёт общей суммы расходов за любой указанный пользователем период.
*   **Сохранение данных:** вся информация надёжно хранится в локальном файле `data.json`.
*   **Загрузка данных:** при запуске программа автоматически загружает ранее сохранённые расходы.
*   **Валидация:** защита от ввода некорректных данных (отрицательные суммы).

## Технологии

*   **Python:** основной язык программирования.
*   **Tkinter:** стандартная библиотека Python для создания графического интерфейса.
*   **tkcalendar:** расширение для удобного выбора дат.
*   **JSON:** формат файла для хранения данных.
*   **Git:** система контроля версий для отслеживания изменений в коде.
